# Username Checker
Use this project to check whether username is available on various popular websites

## [Demo] (https://daksh7011.com/demo/existance/)

# Supported sites
* Instagram
* Disqus
* Pinterest
* Codepen
* JsFiddle
* Twitter
* Patreon
* Youtube
* Imgur
